MIKE & SONS ROOFING — LOGO PACK
================================

COLORS
  Blue   #2B5BA7   (primary)
  Red    #C8332B   (accent)
  Navy   #15233D   (ink / text)
  Light blue #3E73C4 (shield border on dark)

FONTS
  Archivo         — wordmark (800 / 900)
  JetBrains Mono  — tagline / small data (700)
  Google Fonts: https://fonts.google.com/specimen/Archivo

FILES
  SVG/  (vector — infinitely scalable, preferred for print & web)
    symbol-color.svg        shield mark, full color
    symbol-reverse.svg      shield mark for dark backgrounds
    symbol-mono-navy.svg    single-color version (stamps, invoices, fax)
    favicon.svg             rounded app/site icon
    lockup-horizontal.svg   mark + MIKE & SONS / ROOFING + tagline
    lockup-stacked.svg      mark above the wordmark
  PNG/  (raster — transparent background unless noted)
    symbol-color-512 / 1024
    symbol-reverse-1024
    favicon-64 / 256
    lockup-horizontal-2560
    lockup-stacked-1520
    lockup-reverse-onnavy-2560 (on navy tile)

USAGE
  • Keep clear space around the mark equal to the height of the shield's window.
  • Don't recolor, stretch, rotate, or add effects to the mark.
  • Use the reverse version on dark photos or navy/blue backgrounds.
  • Minimum size for the shield mark: 24px tall on screen.

Symbols are font-free vectors and render anywhere. The text lockups use Archivo —
install it (free, Google Fonts) to edit the SVGs, or use the PNGs as-is.
